Photos du menu Le Pacha 2.

Depose ici les images des plats avec les noms indiques dans `data/menu.json`
(`photoFile`). Exemple :

- `menu-sandwich-grec.jpg`
- `menu-assiette-mixte.jpg`
- `tacos.jpg`

Formats conseilles : `.jpg` ou `.webp`, ratio horizontal ou carre, lumiere chaude,
plat bien visible. La page `/menu` affiche automatiquement la photo si le fichier
existe, sinon elle garde un visuel de remplacement.
